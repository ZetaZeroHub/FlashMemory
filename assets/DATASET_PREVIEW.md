## 六、关键提醒(看 README §6-§8 完整规则)

1. ⚠️ 错拒应答样本 → **整道题 0 分**(不是只丢 15%)
2. ⚠️ `must_not_contain` 任一命中 → **关键词得分 = 0**
3. ⚠️ 全拒答策略 → **跨任务 Precision 暴跌**,陷阱反而扣分
4. ⚠️ 答案残留 `{{TEMPLATE}}` 占位符 → 流畅性扣分
5. ✅ 多召回比少召回好(F1@3 看完整性)
